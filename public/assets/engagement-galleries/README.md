# Engagement galleries

Add project images to one of these folders and the matching gallery page will update automatically on `dev`, `build`, `typecheck`, and `lint`:

- `public/assets/engagement-galleries/imi`
- `public/assets/engagement-galleries/siemens-energy`
- `public/assets/engagement-galleries/1590-energy`

Supported file types:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.avif`
- `.svg`

Images are sorted by filename, so naming files like `01.webp`, `02.webp`, `03.webp` keeps the gallery order stable.
